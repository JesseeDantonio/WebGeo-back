
## Description

Un projet web destiné à visualiser, gérer des données géospatiale. Ceci est la partie Back-end.

## Dépendances

- [Node.js](https://nodejs.org/en)
- [Docker](https://docs.docker.com/engine/install/)


## Installation

```bash
$ npm install

$ docker compose build

$ docker compose up
```
## Running the app

```bash
# production
$ npm run start
```
